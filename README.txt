SISTEMA DE BUSCA E OFERTA DE EMPREGOS - FRONTEND

Projeto desenvolvido para a disciplina Tecnologias Cliente Servidor - UTFPR
Curso: Analise e Desenvolvimento de Sistemas

TECNOLOGIAS UTILIZADAS:
- Next.js
- React

COMO EXECUTAR O PROJETO:

1) Instalar dependências
   npm install
   (ou: yarn install / pnpm install)

2) Configurar variáveis de ambiente
   Crie um arquivo .env.local na raiz do projeto com o seguinte conteúdo:
   
   NEXT_PUBLIC_API_URL=http://localhost:3333
   
   Nota: Substitua http://localhost:3333 pela URL do seu servidor backend

3) Iniciar o servidor em modo desenvolvimento
   npm run dev
   (ou: yarn dev / pnpm dev)

4) Acessar a aplicação
   Abra seu navegador e acesse:
   http://localhost:3000

CONFIGURAÇÃO DO SERVIDOR:

1) Ao acessar o aplicativo, clique no botão "Servidor"
2) Digite o endereço do seu backend (ex: http://localhost:3000)
3) Clique em "Salvar alterações"

PRÉ-REQUISITOS:

- Node.js (versão 20 ou superior)
- npm, yarn, pnpm
- Git
